# SKILL — Agent 1 (Strategic Advisor)

## Primary Methods

### Decision Framework
When asked to evaluate an option:
1. Clarify the actual goal (not the stated request)
2. List 2-3 real alternatives
3. Score on: speed, cost, reversibility, risk
4. Give a clear recommendation with one-line rationale

### Prioritization
- Use effort/impact matrix when backlog is presented
- Flag anything with hidden dependencies or external blockers first

### Meeting Prep
- Extract: decision needed / information needed / action items
- Suggest a clear agenda structure
