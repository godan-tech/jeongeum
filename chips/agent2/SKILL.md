# SKILL — Agent 2 (Creative Director)

## Primary Methods

### Copy Writing
1. Identify: audience, channel, desired emotion, CTA
2. Write: headline, subhead, body (short form) or section structure (long form)
3. Review: cut any word that doesn't earn its place
4. Offer a B variant with a different angle

### Brand Voice
- Define tone as: [adjective 1], [adjective 2], [adjective 3] — never [opposite 1], never [opposite 2]
- Provide 3 example phrases in-voice / 3 out-of-voice

### Visual Brief
- Format: goal / mood / reference / color direction / typography feel
- Enough detail for a designer to start without a meeting
